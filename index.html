Projects

VANET (2022)

Designed and implemented a sophisticated Vehicular Ad-Hoc Network (VANET) system, finely tuned to optimize Emergency Medical Services (EMS) with a primary focus on ambulance assistance. The project's core objective is to elevate emergency vehicle communication by seamlessly transmitting automatic alerts enriched with precise Global Positioning System (GPS) coordinates and optimal route details in real-time.

In routine scenarios, the system employs Vehicle-to-Vehicle (V2V) and Vehicle-to-Infrastructure (V2I) communication utilizing Dedicated Short-Range Communications (DSRC) and Cellular-Vehicle-to-Everything (C-V2X) technologies. Through GeoNetworking protocols, Cooperative Awareness Messages (CAM) are disseminated to nearby vehicles, fostering preemptive awareness and ensuring proactive hazard mitigation for a safer driving environment.

During high-traffic situations, the system dynamically applies sophisticated Intelligent Transportation System (ITS) techniques such as Dynamic Route Guidance (DRG) and Traffic Adaptive Routing (TAR), leveraging advanced Traffic Flow Optimization (TFO) algorithms. These algorithms, informed by real-time traffic data obtained through simulation using OpenModelica for microsimulation and SUMO for macroscopic traffic flow analysis, enable dynamic rerouting of emergency vehicles to circumvent congestion hotspots and minimize response time.

The implementation of this system was orchestrated through the integration of cutting-edge open-source simulation platforms, including OMNET++ for network modeling and simulation, and SUMO for traffic simulation and analysis. By harnessing the power of these advanced tools and protocols, the project not only achieved robust functionality but also ensured scalability, interoperability, and adaptability for deployment in real-world scenarios.
